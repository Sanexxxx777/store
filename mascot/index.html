Living Mascot

Страница изображает «ваш сайт» в двух темах. Персонаж читает цвета из CSS-токенов и перекрашивается сам — тот же движок и тот же контракт, что приедут к вам в ZIP.

кликните по персонажу · подразните его пятью кликами · оставьте в покое на 50 секунд

эмоции — вызываются и из вашего кода: mascot.emote('wink')
тема хоста — персонаж перекрашивается сам
цвета вживую — контракт opts.colors()

Эти пикеры дергают тот же API, которым ваш сайт свяжет персонажа со своей палитрой. Ничего пересобирать не нужно.

подключение — целиком 
<!-- ESM: -->
import { createLivingMascot } from './mascot.esm.js';
const m = createLivingMascot(canvas, { colors: () => ({ a: '#7fb3ff', b: '#5a86d6' }) });

<!-- или обычным скриптом: -->
<script src="mascot.min.js"></script>
window.LivingMascot(canvas, { colors: ... });

Canvas 2D, ноль зависимостей, 6.9 КБ gzip. Проверено: desktop, mobile, light/dark, reduced-motion.

Стартовая цена, личная лицензия. В ZIP: исходники, обе сборки (ESM + script), это демо и лицензия — отправляю после оплаты в переписке.
