# Security Policy

This is a static storefront (no backend, no user data storage, no auth). Contact form submissions go through a client-side `mailto:` flow.

## Reporting a vulnerability

If you find a security issue (e.g. XSS, dependency vulnerability in `assets/octopus3d.js`), please report it privately:

- Telegram: [@Aleksandr_NFA](https://t.me/Aleksandr_NFA)
- Email: sanexxx777@gmail.com

Please do not open a public issue for security reports. I'll acknowledge within a few days.
